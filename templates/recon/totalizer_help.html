Get Totalizer Data

The scraper runs fully automatically in the background — no browser window, no CAPTCHA to solve.

What happens when you tap Get Totalizer:

  1. The daily scraper launches silently in the background
  2. It opens IRAS, reads the CAPTCHA image, and solves it using AI
  3. It downloads Price, Shift Totalizer, and ISS boundary data for the selected date
  4. Results are saved directly to the database
  5. Come back to this page and refresh — Step 1 will show green

Timing

The scraper typically takes 1–2 minutes. If it doesn't show green after 3 minutes, check instance/scraper.log for errors.

Note

The scraper needs readings for both the target date AND the previous day (opening + closing totalizers). Both are fetched automatically in a single run.
